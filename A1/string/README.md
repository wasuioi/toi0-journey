# A1-001

### แนวคิด
ใช้ **ตำแหน่งของ string** เพื่อเข้าถึงตัวอักษรตัวแรกและตัวที่สอง (string คือ char ที่เก็บข้อมูลแบบ arrays)

### ประสิทธิภาพ (complexity)
* **Time Complexity:** $O(n)$ เพราะรับค่า sting ใช้เวลาตามจำนวนตัวอักษร
* **Space Complexity:** $O(n)$ เพราะเป็นพื้นที่สำหรับความยาวตัวอักษรทั้งสอง string
