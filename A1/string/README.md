# A1-001

### แนวคิด (Approach)
โจทย์นี้เป็นการรับค่าสตริง 2 ตัว แล้วนำมาแสดงผลร่วมกับข้อความ "Hello" รวมถึงการเข้าถึงดัชนี (Index) ของสตริงเพื่อดึงตัวอักษร 2 ตัวแรกของแต่ละคำออกมา

### ประสิทธิภาพ (complexity)
* **Time Complexity:** $O(n + m)$ 
    * โดยที่ $n$ คือความยาวของสตริง `a` และ $m$ คือความยาวของสตริง `b` 
    * เหตุผล: การอ่านค่า (cin) และการพิมพ์ค่า (cout) สตริงทั้งหมดต้องไล่ไปทีละตัวอักษรตามความยาวของข้อมูล
* **Space Complexity:** $O(n + m)$
    * เหตุผล: เราต้องใช้หน่วยความจำในการเก็บตัวแปรสตริงทั้งสองตัวตามขนาดที่รับเข้ามา

### ทำไมถึงเลือกวิธีนี้?
เนื่องจากโจทย์เป็นการจัดการสตริงเบื้องต้น การเข้าถึง Index อย่าง `a[0]` หรือ `a[1]` นั้นทำได้รวดเร็วมาก ($O(1)$) การใช้ `cin/cout` มาตรฐานจึงเพียงพอต่อความต้องการ
